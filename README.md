# 🛠️ IT Helpdesk Diagnostic and Repair Utility

A Windows-based troubleshooting utility developed using **Batch Scripting** to simplify common IT support and system maintenance tasks. The tool helps diagnose system issues, repair Windows components, perform health checks, and resolve software and network-related problems through a user-friendly menu-driven interface.

---

## 📖 Overview

While learning **Windows Administration** and **IT Support** concepts, I developed this utility to automate several troubleshooting tasks commonly performed by **Helpdesk** and **Desktop Support Engineers**.

The primary objective was to reduce manual effort, improve troubleshooting efficiency, and provide a centralized solution for system diagnostics, repair, and maintenance.

## 🚀 How It Works

The utility provides multiple troubleshooting options through an interactive menu system. Users can select individual diagnostic or repair functions based on the issue they are experiencing.

### 🔍 Quick Health Check

Checks the health status of the Windows image and identifies potential corruption issues.

### 🩺 Advanced System Scan

Performs a detailed analysis of system health to detect integrity and performance-related problems.

### 🔧 Windows Repair

Uses built-in Windows repair tools to restore damaged system components and improve system stability.

### 📂 System File Verification

Scans protected operating system files and automatically repairs corrupted files when possible.

### 🔄 Windows Update Troubleshooting

Resets update-related services and components to resolve common Windows Update failures.

### ⚡ Automated Repair Sequence

Runs a complete troubleshooting workflow consisting of:

1. 🔍 System Scan
2. 🔧 Component Repair
3. 📂 File Verification

This process follows Microsoft's recommended repair methodology.
